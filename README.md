# Taskana app

## Установка зависимостей

Перед первым запуском проекта установите все необходимые зависимости:

```
npm install
```
или

```
npm i
```

## Запуск проекта в dev режиме и открытие в дефолтном браузере

```
npm run dev
```
## Сборка проекта в build версию

```
npm run build
```
