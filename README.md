# Taskana app

## Установка зависимостей

Перед первым запуском проекта установите все необходимые зависимости:

```
npm install
```

## Запуск проекта в dev режиме и открытие в браузере

```
npm run dev
```
## Сборка проекта в build версию

```
npm run build
```

classNames тут - это утилитарная функция, а не библиотека. Так что все по тз :)
