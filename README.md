# Taskana app

## Установка зависимостей

Перед первым запуском проекта установите все необходимые зависимости:

```bash
npm install
```

## Запуск проекта в dev режиме

```bash
npm run dev
```
## Сборка проекта в build версию

```bash
npm run build
```

Для тестирования loading состояния кнопок надо в компоненте Button.jsx заменить дефолтное значения пропса isLoading на true. Специально не стал убирать возможность с ними взаимодействовать в этом состоянии для удобства тестирования.

Оставил несколько комментариев в папке фидбэка для последних проверяющих.
